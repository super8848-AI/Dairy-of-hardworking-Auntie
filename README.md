# 鹅腿阿姨创业日记 (Dairy of Hardworking Auntie)

摆摊小游戏：「鹅腿阿姨」推着三轮小车，在北京大学门口卖「鹅腿」。
你将亲手体验一根 ¥1.5 的冷冻鸭腿，如何经过滚揉、卤制、膨胀，变成 ¥16 的「正品鹅腿」。

> 本游戏纯属讽刺：鸭腿再肥，也变不成鹅腿。

## 玩法流程

1. **菜市场进货**：选择鸭腿（¥1.5/根）还是正品鹅腿（¥15/根）——选鹅腿会被灵魂拷问
2. **开面包车回家**：冻腿驮回昌平区小别野
3. **深度加工**：真空滚揉吸盐水 → 老卤加葱卤制 → 成品膨胀 20%~50%
4. **北大门口开卖**：每根 ¥16，左上角计数板实时统计收益（人民币）
5. **隐藏剧情**：有同学发现鹅腿是绿色的……
6. **卖出 20 根**：成就「北大演讲」——声望 +10000，人流量 +1000%，同学开始排队
7. **卖出 30 根**：解锁可玩副本「国贸 CBD」（写字楼背景、白领顾客、质疑 NPC）
8. **国贸第三次被质疑**：阿姨逃走，解锁新副本「我没有对不起任何人」（敬请期待）
